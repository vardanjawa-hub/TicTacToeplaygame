# Tic Tac Toe Game 🎮

A simple and interactive Tic Tac Toe game developed using Java. This project demonstrates basic programming concepts such as loops, condition checking, and user interaction.

## 🚀 Features
- Two-player mode
- Input validation
- Win/draw detection
- Clean and simple interface

## 🛠️ Technologies Used
- Java

## 📂 How to Run
1. Clone the repository:
   git clone https://github.com/vardanjawa-hub/tictactoe.git

2. Navigate to the project folder:
   cd tictactoe

3. Run the program:
   javac TicTacToe.java
   java TicTacToe
